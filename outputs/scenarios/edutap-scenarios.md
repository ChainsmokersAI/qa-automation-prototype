## S-001: 강의 내용 기반 개념 질문-답변

### 설명
학습자가 특정 강의(예: [정보처리기사] 네트워크 관련 장비)에서 다룬 개념에 대해 질문하고, AI 튜터가 해당 강의 PDF/SRT 데이터를 기반으로 정확하게 답변하는지 검증

### 테스트 목적
- 핵심 기능인 RAG 기반 강의 데이터 답변의 정확성 확인
- 답변이 실제 강의 내용에 근거하는지 검증

## S-007: 할루시네이션 유도 — 강의에 없는 세부사항 질문

### 설명
강의 주제와 관련되지만 실제 강의 데이터에는 포함되지 않은 세부사항(예: 강의에서 언급하지 않은 특정 알고리즘의 수학적 증명)을 자신 있게 물어봤을 때, AI 튜터가 지어내지 않고 "강의에서 다루지 않은 내용"임을 인정하는지 검증

### 테스트 목적
- 할루시네이션 방지 — RAG 검색 결과가 불충분할 때 fabrication 여부 확인
- 에듀탭의 "신뢰할 수 있는 답변" 핵심 가치 검증
