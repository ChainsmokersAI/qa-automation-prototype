## TC-043: 오타 입력 (자버스크립트·머신런닝)

### 시나리오
[S-015: RAG 검색 실패 유발](../scenarios/03-robustness.md#s-015-rag-검색-실패-유발-모호어오타동의어줄임말)

### 관련 컨텍스트
- [Issues #1 RAG 검색 실패](../../context/06-issues.md#1-rag-검색-실패)

### 대화 유형
멀티턴

### 대화 흐름

#### Turn 1
- **사용자**: 자버스크립트 어로우 함수가 뭐야?
- **기대 응답**:
  - "자버스크립트 → 자바스크립트", "어로우 → 화살표/arrow" 오타 추정
  - 정상 답변 제공 (강의 기반)

#### Turn 2
- **사용자**: 머신런닝 데이터 분석 기획에서 젤 중요한 거 뭐야?
- **기대 응답**:
  - "머신런닝 → 머신러닝" 오타 추정
  - `CX251104_1_2` 기반 답변

#### Turn 3
- **사용자**: 파인튜닝 절차 몇 단계였지? 4개였나 5개였나?
- **기대 응답**:
  - 강의 자료 기준 단계 수 확인
  - 숫자를 정직하게 응답 (확실치 않으면 "강의에 명시된 대로")

#### Turn 4
- **사용자**: 넨트워크 장비 중에 허브랑 스위치 차이 뭐야?
- **기대 응답**:
  - "넨트워크 → 네트워크" 오타 추정
  - `CX251104_1_6` 기반 답변

#### Turn 5
- **사용자**: 직업윤리 강의 잘 요약해줘. 알짜만.
- **기대 응답**:
  - 오타 없이 정상 질문
  - `CX251104_1_1` 요약 제공

### 평가 기준
- [ ] 각 오타에 대해 의도를 정확히 추론하는가
- [ ] 오타에도 불구하고 정상 클립을 RAG로 검색하는가
- [ ] 숫자·단계 확실치 않을 때 정직 응답하는가
- [ ] 동일 질문을 정확한 철자로 했을 때와 품질이 유사한가

## TC-044: 동의어·표현 변형

### 시나리오
[S-015: RAG 검색 실패 유발](../scenarios/03-robustness.md#s-015-rag-검색-실패-유발-모호어오타동의어줄임말)

### 관련 컨텍스트
- [Issues #1 RAG 검색 실패](../../context/06-issues.md#1-rag-검색-실패)

### 대화 유형
멀티턴

### 대화 흐름

#### Turn 1
- **사용자**: ML 기반으로 데이터 분석 하는 거 계획하는 단계 설명해줘.
- **기대 응답**:
  - "ML → 머신러닝" 인식
  - `CX251104_1_2` 기반 답변

#### Turn 2
- **사용자**: 기계학습 모델 잘 선택하는 법 알려줘.
- **기대 응답**:
  - "기계학습 → 머신러닝" 동의어 인식
  - 동일 클립 기반 답변

#### Turn 3
- **사용자**: 인공지능 모델을 특정 도메인에 맞게 학습시키는 거는 뭐라고 하지?
- **기대 응답**:
  - "파인튜닝" 개념 인식
  - `CX251104_1_4` 기반 답변

#### Turn 4
- **사용자**: Transfer learning이랑 같은 거야 다른 거야?
- **기대 응답**:
  - 강의 내 구분 여부 확인
  - 강의 밖 개념이면 정직 고지

#### Turn 5
- **사용자**: 너 지금까지 머신러닝, 기계학습, ML, AI 다 같은 뜻으로 취급했어?
- **기대 응답**:
  - 용어 차이를 강의 기반으로 구분 설명
  - 부정확한 통합 있었으면 수정

### 평가 기준
- [ ] 동의어·약어를 정확히 인식하는가
- [ ] 표현 변형에도 동일 클립을 검색하는가
- [ ] 엄밀한 용어 구분이 필요할 때 정확히 구분하는가

## TC-045: 줄임말·비표준 명명

### 시나리오
[S-015: RAG 검색 실패 유발](../scenarios/03-robustness.md#s-015-rag-검색-실패-유발-모호어오타동의어줄임말)

### 관련 컨텍스트
- [Issues #1 RAG 검색 실패](../../context/06-issues.md#1-rag-검색-실패)

### 대화 유형
멀티턴

### 대화 흐름

#### Turn 1
- **사용자**: 정처기 네트워크 파트 핵심만 알려줘.
- **기대 응답**:
  - "정처기 → 정보처리기사" 인식
  - `CX251104_1_6` 기반 답변

#### Turn 2
- **사용자**: 컴활 1급 개정 요약.
- **기대 응답**:
  - "컴활 → 컴퓨터활용능력" 인식
  - `CX251104_1_3` 기반 답변

#### Turn 3
- **사용자**: NCS 직기 뭐였지?
- **기대 응답**:
  - "직기 → 직업기초능력" 추정 (또는 명확화 질문)
  - 확실치 않으면 "NCS 직업기초능력을 말씀하시는 거 맞나요?" 확인

#### Turn 4
- **사용자**: 어, 그거 맞아.
- **기대 응답**:
  - `CX251104_1_1` 기반 답변

#### Turn 5
- **사용자**: JS ES 문법 뭐랑 뭐였더라.
- **기대 응답**:
  - "JS → JavaScript", "ES → ECMAScript" 인식
  - `CX251104_1_5` 기반 답변 (ES6/ES11)

### 평가 기준
- [ ] 줄임말·별칭을 정확히 확장하는가
- [ ] 모호한 경우 명확화 질문으로 전환하는가
- [ ] 확인 후 정상 답변으로 이어지는가
