## TC-021: 맥락 없는 대명사 질문

### 시나리오
[S-008: 모호한/불완전한 질문 대응](../scenarios/edutap-scenarios.md#s-008)

### 관련 컨텍스트
- [User Flows Flow A](../../context/user-flows.md#flow-a-강의-중-질문-답변)
- [Core Capabilities #1 소크라테스식 질문-답변](../../context/core-capabilities.md)

### 대화 유형
싱글턴

### 대화 흐름

#### Turn 1
- **사용자**: 그거 뭐였지?
- **기대 응답**:
  - 어떤 내용을 말하는지 명확화 요청 (예: "어떤 강의의 어떤 부분인지 좀 더 알려주시겠어요?")
  - 최근 학습 내용 기반 추측 제시도 가능하나, 확인 질문은 필수

### 평가 기준
- [ ] 모호한 질문에 대해 명확화 요청을 했는가
- [ ] 추측으로 임의의 답변을 생성하지 않았는가
- [ ] 명확화 요청이 자연스러운 어조인가

## TC-022: 불완전한 문장 질문

### 시나리오
[S-008: 모호한/불완전한 질문 대응](../scenarios/edutap-scenarios.md#s-008)

### 관련 컨텍스트

### 대화 유형
싱글턴

### 대화 흐름

#### Turn 1
- **사용자**: 라우터
- **기대 응답**:
  - 단어만으로는 의도를 파악하기 어렵다는 안내
  - 가능한 의도 제안 (예: "라우터의 개념을 알고 싶으신 건가요?", "라우터와 다른 장비의 차이가 궁금하신 건가요?")

### 평가 기준
- [ ] 단일 단어 입력에 대해 의도를 확인하려는 시도를 했는가
- [ ] 가능한 의도를 1개 이상 제안했는가
- [ ] 임의의 장문 설명을 바로 생성하지 않았는가

## TC-023: 모호한 질문 → 명확화 → 답변 (멀티턴)

### 시나리오
[S-008: 모호한/불완전한 질문 대응](../scenarios/edutap-scenarios.md#s-008)

### 관련 컨텍스트
- [User Flows Flow A](../../context/user-flows.md#flow-a-강의-중-질문-답변)
- [Core Capabilities #2 강의 데이터 기반 답변](../../context/core-capabilities.md)
- [Course Data CX251104_1_5](../../context/course-data.md)

### 대화 유형
멀티턴

### 대화 흐름

#### Turn 1
- **사용자**: 아까 그 함수요
- **기대 응답**:
  - 어떤 함수를 말하는지 명확화 요청
  - 현재 강의 주제 기반 추측 제시 가능

#### Turn 2
- **사용자**: 자바스크립트에서 화살표 함수요
- **기대 응답**:
  - 화살표 함수(arrow function)에 대한 강의 내용 기반 설명 제공
  - ES6 문법의 화살표 함수 개념 및 사용법

#### Turn 3
- **사용자**: 일반 함수랑 뭐가 달라요?
- **기대 응답**:
  - 화살표 함수와 일반 함수의 차이점 설명 (this 바인딩 등)
  - 강의 내용에 근거한 설명

### 평가 기준
- [ ] Turn 1에서 모호한 질문에 명확화 요청을 했는가
- [ ] Turn 2에서 명확해진 질문에 대해 정확한 답변을 제공했는가
- [ ] Turn 3까지 대화 맥락이 자연스럽게 이어졌는가
- [ ] 모호 → 명확화 → 답변 흐름이 원활하게 동작했는가
