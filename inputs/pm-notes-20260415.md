## API Spec (2026-04-15, 테크리드 제공)

### Endpoint
- **Staging**: `POST https://tapapistaging.coxwave.link/api/v1/chats/stream`
- **Demo site**: https://demo-edutap-staging.vercel.app/

### 인증
- Header: `TAP-API-KEY: <REDACTED>`

### 요청 포맷 (JSON body)
```json
{
  "user_info": {"user_id": "test-user-align"},
  "chat_session_id": "698d62441ee71b05fb65c92d",
  "text": "[사용자 질의]",
  "answer_style": "concise",
  "command_id": "",
  "course_id": "CX251104",
  "course_name": "",
  "course_category": "",
  "course_sub_category": "",
  "clip_id": "[CLIP_ID]",
  "clip_play_head": 0,
  "is_retry": false
}
```

### 응답 포맷 (SSE 스트림)
- `Content-Type: text/event-stream`
- 각 이벤트는 `data: {json}` 라인
- 주요 이벤트 유형:
  - `role=assistant, content={"type":"chat_session_id","value":"..."}` — 세션 ID 확인
  - `role=assistant, content="텍스트 조각", content_type="default"` — 본문 텍스트 청크 (누적하여 전체 답변 조립)
  - `role=reference, content={type:"script", clip_id, location, summary, score, ...}` — RAG 참조 클립 메타데이터
- 스트림 종료는 연결 종료로 판단

- `CX251104_1_2` : [빅데이터 분석] 머신러닝 기반 데이터 분석 기획
- `CX251104_1_3` : [EBS] 컴활 1급 2024 최신 개정 사항 특강
- `CX251104_1_4` : [인공지능] 파인튜닝을 통한 KPoEM 모델 구현
