## 잠재 리스크 (추정)

### 1. RAG 검색 실패
- 공개된 검색 정확도 93.8% → 나머지 **6.2%의 실패 케이스** (무관 청크 검색, 관련 청크 누락)가 오답의 주요 원인이 될 수 있음 [추측: https://zdnet.co.kr/view/?no=20250923141435 (2026-04-15); https://platum.kr/archives/271365 (2026-04-15)]
- 용어 모호성, 동의어, 줄임말, 오타에 대한 검색 품질

### 2. 강의 맥락 벗어난 질문 처리
- 강의 외 주제 질문 시 **할루시네이션 vs 거부**의 경계 판단
- "강의에 안 나온 최신 동향" 요청 시 정직한 한계 고지 여부

### 3. 멀티 클립 교차 질의
- 여러 클립에 걸친 질문에서 어떤 클립을 우선 참조하는가
- 클립 간 상충 정보가 있을 때 처리 (예: JavaScript ES6 vs 다른 클립의 legacy 문법)

### 4. 소크라테스식 응답 피로도
- 항상 질문으로 되돌리면 학습자가 답답함을 느낄 수 있음
- 상황 판단 정확도 (직답이 필요한 사실 질문 vs 사고 유도가 필요한 개념 질문)
