# User Flows (유저 플로우)

공식 사용 가이드는 공개되지 않아, 서비스 설명·포지셔닝 + 에듀탭 연동 대상 과정 `CX251104`의 특성을 결합해 유저 플로우를 추정한다. User Logs가 추후 확보되면 재정밀화 대상.

## Flow 1. 강의 시청 중 개념 질문 (핵심 플로우)
1. 수강생이 강의 영상을 시청 (예: `CX251104_1_2` 머신러닝 기반 데이터 분석 기획)
2. 특정 개념·용어가 이해되지 않음 → 챗봇 호출
3. 챗봇이 현재 재생 중인 클립 맥락을 인지한 상태에서 답변 [추측: https://edutap.ai/ (2026-04-15)]
4. 수강생이 심화 질문 (예시 요청, 비유 요청, "좀 더 쉽게" 요청)
5. 챗봇이 소크라테스식으로 질문을 되돌려 이해 점검 [추측: https://platum.kr/archives/271365 (2026-04-15)]

## Flow 2. 복습 퀴즈 요청
1. 수강생이 시청한 클립에 대해 "퀴즈 내줘" 요청
2. 챗봇이 강의 내용 기반 퀴즈 생성 [추측: https://platum.kr/archives/271365 (2026-04-15)]
3. 수강생 답변 → 정답·해설 제공 → 오답 시 관련 강의 구간 안내

## Flow 3. 여러 클립 교차 참조
1. 수강생이 한 클립 시청 후 다른 클립과의 연관 내용 질문 (예: "머신러닝 데이터 분석 기획"과 "KPoEM 파인튜닝"의 차이)
2. 챗봇이 두 클립의 내용을 RAG로 조회해 비교 설명

## Flow 4. 코드/실습 질문 (JavaScript · AI 파인튜닝 클립)
1. 수강생이 `CX251104_1_5` 자바스크립트 ES6/ES11 혹은 `CX251104_1_4` KPoEM 파인튜닝 클립 시청 중
2. 코드 스니펫 이해·수정 질문
3. 챗봇이 강의 내 예시 기반 설명 + 추가 예시 코드 제공

## Flow 5. 자격증/시험 대비 질문 (컴활·정보처리기사 클립)
1. 수강생이 `CX251104_1_3` 컴활 1급 또는 `CX251104_1_6` 정보처리기사 네트워크 클립 시청 중
2. "이게 시험에 어떻게 나오나요?", "요약해줘" 등 시험 대비 요청
3. 챗봇이 강의 범위 내에서 요약 + 예상 문제 제시

## Flow 6. 학습 진도 점검
1. 수강생이 "내가 지금까지 뭘 배웠지?" / "이해도 점검해줘" 질문
2. 챗봇이 학습 현황 데이터 기반으로 요약·점검 [추측: https://edutap.ai/ (2026-04-15)]

## Flow 7. 범위 밖 질문 (경계 플로우)
1. 수강생이 강의에 없는 주제 질문 (예: 일상 잡담, 수강료 문의, 최신 뉴스)
2. 챗봇이 범위 밖임을 안내하거나 학습 관련 주제로 유도 — 할루시네이션 vs 거부 경계 테스트 핵심 영역
