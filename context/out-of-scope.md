# Out-of-Scope

- **강의 데이터 범위 밖 질문**: 일반 상식, 시사, 비학습 잡담 등 → 강의 자료에 없는 내용은 답변 거부 또는 범위 안내 예상 [추측: 에듀탭이 RAG 기반 강의 데이터 한정 답변을 강조하는 점에서 추론]
- **실시간 강사 연결**: 인간 강사와의 직접 상담/연결 기능 [추측: 도메인 일반 — AI 튜터 플랫폼은 통상 강사 실시간 연결 미지원]
- **학습자 간 소통**: 커뮤니티, 토론방, 그룹 학습 기능 [추측: 1:1 AI 튜터에 특화된 서비스이므로 소셜 기능 부재 추론]
- **수강신청/결제**: 행정·결제 업무 처리 [추측: AI 튜터 챗봇의 범위 밖으로 추론]
- **코드 실행/실습 환경**: 프로그래밍 강의라도 코드를 직접 실행하는 IDE/샌드박스 기능 [추측: SDK 연동 AI 튜터 특성상 실행 환경 미포함 추론]
