# User Flows

## Flow A. 강의 중 질문-답변

> 강의 시청 → 궁금한 점 발생 → AI 튜터에게 질문 → 소크라테스식 역질문/힌트 → 학습자 사고 → 답변 확인/심화 설명

## Flow B. 복습 퀴즈 루프

> 강의 시청 완료 → 복습 퀴즈 제공 → 학습자 응답 → 정답/오답 피드백 → 취약 부분 재학습 유도

## Flow C. 심화 학습

> 기본 이해 완료 → 관련 심화 질문 → AI 튜터가 강의 자료 기반 추가 설명 → 보충 학습 콘텐츠 추천
